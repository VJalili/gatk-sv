# Module 05/06

Combines variants across multiple batches,
resolves complex variants, re-genotypes,
and performs final VCF clean-up.

## Prerequisites:
* [Module 04](04)
* (Optional) [Module 04b](04#module-04b)

## Inputs:
* RD, PE and SR file URIs ([Module 00c](00c))
* Batch filtered PED file URIs ([Module 03](03))
* Genotyped PESR VCF URIs ([Module 04](04))
* Genotyped depth VCF URIs ([Module 04](04) or [Module 04b](04#module-04b))
* SR pass variant file URIs ([Module 04](04))
* SR fail variant file URIs ([Module 04](04))
* Genotyping cutoff file URIs ([Module 04](04))
* Batch IDs
* Sample ID list URIs

## Outputs:
* Finalized "cleaned" VCF and QC plots
